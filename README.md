# Task Manager API 📝

A simple backend Task Management system built in Java to practice object-oriented programming, clean code principles, and REST-style application design.

## 🚀 Features
- Create new tasks
- View all tasks
- Update task status
- Delete tasks
- Task prioritization
- Due date management

## 🛠 Tech Stack
- Java
- OOP Principles
- Collections Framework
- Maven
- REST Concepts
- JSON data handling

## 📂 Project Structure
src
├── model → Task entity
├── service → Business logic
├── repository → Data storage
├── controller → Application interaction
└── Main.java → Application entry point

## 🧠 Concepts Practiced
- Object-Oriented Programming
- Encapsulation & Abstraction
- Separation of Concerns
- CRUD Operations
- Clean Code Practices
